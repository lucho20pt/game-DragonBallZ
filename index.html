Welcome

The DBZ Catch Game is an unofficial and non-profit game that was made by a DBZ fan created by Daniel Batista to learn and test the new potential of creating games with html5.

The Game is made with html5 using the canvas tag with a lot of javascript and jquery implemented.

The game was developed in 2013
It a is very simple game and is still evolving as I have spare time to work

How to Play

This game is to catch up on the hero catching the villain

When harvesting the villain you won a point.

The difficulty is increased when it takes some time to catch the villain, this disappears and pops up somewhere else.

When the clock gets to 0, the game is over and your score will be saved.

You can zoom in (Ctrl +) or zoom out (Ctrl -) to increase the canvas size
